# CVBER — Search Engine Submission Guide

Submit your site to search engines to get indexed faster.

## Google Search Console

1. Go to https://search.google.com/search-console
2. Click "Add Property"
3. Enter: https://cvber.vercel.app
4. Verify via HTML tag or DNS
5. Submit sitemap: https://cvber.vercel.app/sitemap.xml
6. Request indexing for all pages

## Bing Webmaster Tools

1. Go to https://www.bing.com/webmasters
2. Sign in with Microsoft account
3. Add site: https://cvber.vercel.app
4. Submit sitemap: https://cvber.vercel.app/sitemap.xml

## Yandex Webmaster

1. Go to https://webmaster.yandex.com
2. Add site: https://cvber.vercel.app
3. Verify ownership
