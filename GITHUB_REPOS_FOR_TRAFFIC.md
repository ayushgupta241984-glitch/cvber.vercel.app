## Quick Reference: What to Install First

```bash
# Priority 1 — SEO foundation
npm install next-seo next-sitemap

# Priority 2 — Content
npm install @next/mdx @mdx-js/loader @mdx-js/react

# Priority 3 — GEO tracking (Python)
pip install gego  # or clone AI2HU/gego

# Priority 4 — Social media (self-host)
# Clone gitroomhq/postiz-app and deploy
```
