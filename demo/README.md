# Demo Assets

This folder contains sample assets for demonstrating CVBER's capabilities.

## Contents

| File | Description |
|------|-------------|
| `sample-artwork.png` | Sample public-domain artwork for testing |
| `sample-evidence-report.json` | Example evidence report output |
| `sample-dmca-template.md` | Example DMCA takedown notice |

## Usage

1. Upload `sample-artwork.png` to the CVBER dashboard
2. Review the generated evidence report
3. Download the evidence ZIP
4. Use `sample-dmca-template.md` as a reference for filing

## Notes

- All demo assets are public domain or created for testing purposes
- Do not use real user artwork in demo materials
- Evidence reports are examples, not real legal documents
